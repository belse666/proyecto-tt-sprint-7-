# proyecto-tt-sprint-7-
proyecto del sprint 7


# DEPENDENCIAS 
Python 3.x
Streamlit (Interfaz de usuario)
Pandas (Manipulación de datos)
Plotly / Matplotlib (Visualizaciones)

# Características
Check box : para mostrar tabla de datos. 
Selección Dinámica: Selección de marcas mediante menús interactivos.
Visualización en Tiempo Real: Gráficos que se actualizan según los filtros.

# Instalación y Uso
1. clona el repositorio
   git clone https://github.com
2. Crear un entorno virtual (opcional pero recomendado)
   python -m venv venv
   source venv/bin/activate
3. Instalar dependencias(si es nesesario):
   pip install -r requirements.txt
4. Ejecutar la aplicación:
   streamlit run app.py
